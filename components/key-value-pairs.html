Key Value Pairs In Progress

Key value pairs are used to display data with their corresponding label. Color and alignment are used to enhance readibility allowing users to quickly scan for the page for relevant data.

Best Practices:

  • Labels should correspond with the labels used for data creation and/or editing.
  • Labels appear to the left and are right aligned.
  • Data values are displayed to the right and left aligned.
  • Icons should be used sparingly.
