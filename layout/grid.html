Responsive 12 Column GridIn Progress

Implementation of the responsive fluid grid system is flexible and makes it simple to adjust the grid to custom needs. By default, the grid consists of 12 columns which will stack responsively for tablet and mobile screens. The foundation of the Helix grid is Bootstrap; columns and gutter sizes have been adjusted to optimize the layout of content across multiple devices.

Usage

The grid is intended to be applied to the content area of the page, excluding site navigation.
