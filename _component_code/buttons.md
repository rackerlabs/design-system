### Button Specs

List of associated css values used across the three button sizes.

#### Standard Button Specs

- `font-family:roboto-medium`
- `font-size:14px`
- `text-transform:uppercase`
- `border-radius:2px`

#### Large Button Specs

- `font-family:roboto-medium`
- `font-size:16px`
- `text-transform:uppercase`
- `border-radius:2px`

#### Compact Button Specs

- `font-family:roboto-medium`
- `font-size:10px`
- `text-transform:uppercase`
- `border-radius:2px`
