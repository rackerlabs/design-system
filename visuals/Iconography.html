Icon Font

Icons should be used sparingly and with purpose. Icons can draw attention, help with way-finding or overcome space constraints. Universally understood icons work well. If you think your icon needs a text label to be identifiable, then a text label is probably all you need. Icons can be difficult to get right and need a lot of testing to verify understandability and discoverability.

If you think there is need for an addition icon to be added to the list please submit your request to the Helix Team.
